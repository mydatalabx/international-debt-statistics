# International Debt Statistics Analysis

SQL-based analysis of international debt data to identify patterns, trends, and insights across different countries and debt indicators.

## Overview
This project uses SQL queries to analyze international debt statistics and answer key questions about global debt patterns.

## Technologies Used
- SQL
- Jupyter Notebook (for documentation)
- Database querying and analysis

## Key Insights
- Analysis of debt across multiple countries using SQL queries
- Identification of highest debt indicators
- Aggregation and filtering of international debt data

## Files
- `notebook.ipynb` - SQL queries and analysis
